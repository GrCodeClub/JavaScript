JavaScript Εντολές

Εκτύπωση μηνυμάτων ή τιμών στο console του προγράμματος περιήγησης.

console.log("Hello, world!");

let και const: Χρησιμοποιούνται για τον ορισμό μεταβλητών. Οι μεταβλητές που δηλώνονται με let μπορούν να αλλάξουν τιμή, ενώ οι μεταβλητές που δηλώνονται με const είναι σταθερές και δεν μπορούν να αλλάξουν τιμή.

let count = 10;
const PI = 3.14159;

if-else: Χρησιμοποιείται για τον έλεγχο της συνθήκης και την εκτέλεση κώδικα βάσει αυτής της συνθήκης.

let age = 18;
if (age >= 18) {
    console.log("Είστε ενήλικας.");
} else {
    console.log("Είστε ανήλικος.");
}

for loop: Χρησιμοποιείται για την επανάληψη κώδικα συγκεκριμένο αριθμό φορών.

for (let i = 0; i < 5; i++) {
    console.log(i);
}

while loop: Χρησιμοποιείται για την επανάληψη κώδικα όσο μια συνθήκη είναι αληθής.

let num = 0;
while (num < 5) {
    console.log(num);
    num++;
}

function: Χρησιμοποιείται για τον ορισμό και την κλήση συναρτήσεων.

function greet(name) {
    console.log(`Καλημέρα, ${name}!`);
}
greet("ΓΙΩΡΓΟΣ");

Array: Χρησιμοποιείται για τη δημιουργία και διαχείριση πινάκων.

let colors = ["Κόκκινο", "Πράσινο", "Μπλε"];
console.log(colors[0]); // Κόκκινο

Object: Χρησιμοποιείται για τη δημιουργία και διαχείριση αντικειμένων με ιδιότητες.

let person = {
    firstName: "Γιάννης",
    age: 30
};

let onoma = person.firstName;
let ilikia = person.age;

console.log(onoma); // Γιάννης
console.log(ilikia); // 30
