Libraries - Framework

React.js

React είναι μια βιβλιοθήκη για την ανάπτυξη διαδραστικών και δυναμικών UI (User Interfaces). Χρησιμοποιεί components και Virtual DOM για αποδοτική ανανέωση του περιεχομένου της σελίδας χωρίς πλήρη ανανέωση του DOM.

<script src="https://unpkg.com/react@17/umd/react.production.min.js"></script>
<script src="https://unpkg.com/react-dom@17/umd/react-dom.production.min.js"></script>

Vue.js

Vue.js είναι ένα framework για τη δημιουργία frontend εφαρμογών. Προσφέρει απλή και ευέλικτη προσέγγιση στη δέσμευση δεδομένων και τη δημιουργία δυναμικών διεπαφών χρήστη. Υποστηρίζει two-way data binding και component-based architecture.

<script src="https://cdn.jsdelivr.net/npm/vue@2"></script>

jQuery

jQuery είναι μια βιβλιοθήκη που απλοποιεί την εργασία με το HTML DOM, τα γεγονότα, τις animations, και τις AJAX κλήσεις. Είναι χρήσιμη για γρήγορη ανάπτυξη και συμβατότητα μεταξύ διαφόρων browsers.

<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

Angular

Angular είναι ένα πλήρες framework για την ανάπτυξη web εφαρμογών. Παρέχει ενσωματωμένα εργαλεία για data binding, routing, form validation, και πολλά άλλα. Χρησιμοποιεί TypeScript για την ανάπτυξη.

<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.2/angular.min.js"></script>

Math.js

Math.js είναι μια βιβλιοθήκη για μαθηματικά που υποστηρίζει διάφορους τύπους αριθμητικών υπολογισμών, αλγεβρικές εξισώσεις, και υπολογισμούς με μονάδες μέτρησης. Παρέχει εργαλεία για την εκτέλεση μαθηματικών συναρτήσεων, πίνακες και matrices, και άλλα μαθηματικά εργαλεία.

<script src="https://cdnjs.cloudflare.com/ajax/libs/mathjs/11.0.0/math.min.js"></script>

Algebrapi

Algebrapi παρέχει εργαλεία για την επίλυση αλγεβρικών εξισώσεων και τη διαχείριση αλγεβρικών εκφράσεων. Είναι χρήσιμη για εκπαιδευτικούς και μαθηματικούς που θέλουν να ενσωματώσουν αλγεβρικές λειτουργίες στις εφαρμογές τους.

<script src="https://cdnjs.cloudflare.com/ajax/libs/algebrapi/1.0.0/algebrapi.min.js"></script>

Plotly.js

Plotly.js είναι μια βιβλιοθήκη για τη δημιουργία δυναμικών και αλληλεπιδραστικών γραφημάτων και visualizations. Υποστηρίζει διάφορους τύπους γραφημάτων, όπως scatter plots, bar charts, line charts, και 3D visualizations.

<script src="https://cdn.plot.ly/plotly-latest.min.js"></script>

MathJax

MathJax είναι μια βιβλιοθήκη που επιτρέπει την εμφάνιση μαθηματικών τύπων και συμβόλων σε ιστοσελίδες. Υποστηρίζει LaTeX, MathML και AsciiMath για την παρουσίαση μαθηματικών εξισώσεων σε HTML.

<script src="https://cdnjs.cloudflare.com/ajax/libs/mathjax/3.2.0/es5/tex-mml-chtml.min.js"></script>
