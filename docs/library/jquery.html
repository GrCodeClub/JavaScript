jQuery

Εισαγωγή στο jQuery

 <script src="https://code.jquery.com/jquery-3.6.0.min.js">

Βασική Σύνταξη

  1. $ είναι μια συντομογραφία για το jQuery.
  2. selector επιλέγει το (τα) στοιχείο (α) HTML.
  3. action είναι η ενέργεια jQuery που θα εκτελεστεί στο επιλεγμένο στοιχείο (α).
$(selector).action();

Επιλογείς

Επιλέγει ένα στοιχείο με μια συγκεκριμένη ID.

$('#myId').css('color', 'red');

Επιλέγει στοιχεία με μια συγκεκριμένη κλάση.

$('.myClass').hide();

Επιλέγει όλα τα στοιχεία ενός συγκεκριμένου τύπου.

$('p').fadeIn();

Γεγονότα

Γεγονός Κλικ

$('#myButton').click(function() {
    alert('Το κουμπί πατήθηκε!');
});

Γεγονός Hover

$('#myDiv').hover(
    function() {
        $(this).css('background-color', 'yellow');
    },
    function() {
        $(this).css('background-color', 'white');
    }
);

Εφέ

Απόκρυψη/Εμφάνιση

$('#myDiv').hide();
$('#myDiv').show();

Fade In/Out

$('#myDiv').fadeIn();
$('#myDiv').fadeOut();

Slide Up/Down

$('#myDiv').slideUp();
$('#myDiv').slideDown();

AJAX

Το jQuery απλοποιεί τις Ajax αιτήσεις με τη μέθοδο $.ajax().

Βασική Κλήση AJAX

$.ajax({
    url: 'https://api.example.com/data',
    method: 'GET',
    success: function(data) {
        console.log(data);
    },
    error: function(error) {
        console.error('Σφάλμα:', error);
    }
});

Διαχείριση DOM

Μπορείτε να χειριστείτε το περιεχόμενο HTML με το jQuery πολύ εύκολα.

Αλλαγή Περιεχομένου

$('#myDiv').html('Νέο περιεχόμενο εδώ!');

Προσθήκη Κλάσης

$('#myDiv').addClass('newClass');

Αφαίρεση Κλάσης

$('#myDiv').removeClass('oldClass');
