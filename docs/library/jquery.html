jQuery

Εισαγωγή στο jQuery

 <script src="https://code.jquery.com/jquery-3.6.0.min.js">

Βασική Σύνταξη

  1. $ είναι μια συντομογραφία για το jQuery.
  2. selector επιλέγει το (τα) στοιχείο (α) HTML.
  3. action είναι η ενέργεια jQuery που θα εκτελεστεί στο επιλεγμένο στοιχείο (α).
$(selector).action();

Επιλογείς

Επιλέγει ένα στοιχείο με μια συγκεκριμένη ID.

$('#myId').css('color', 'red');

Επιλέγει στοιχεία με μια συγκεκριμένη κλάση.

$('.myClass').hide();

Επιλέγει όλα τα στοιχεία ενός συγκεκριμένου τύπου.

$('p').fadeIn();
