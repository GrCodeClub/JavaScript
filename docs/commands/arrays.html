Arrays

Δημιουργία Arrays

Μπορείς να δημιουργήσεις έναν πίνακα χρησιμοποιώντας αγκύλες [].

// Δημιουργία ενός κενών πίνακα
let emptyArray = [];

// Δημιουργία ενός πίνακα με μάρκες αυτοκινήτων
let cars = ["Toyota", "Honda", "BMW"];

Πρόσβαση σε Στοιχεία Πίνακα

Η πρόσβαση σε στοιχεία του πίνακα γίνεται με χρήση του index

console.log(cars[0]); // Εκτυπώνει: Toyota
console.log(cars[1]); // Εκτυπώνει: Honda
console.log(cars[2]); // Εκτυπώνει: BMW

Μήκος του Πίνακα

Μπορείς να βρεις το μήκος (αριθμό στοιχείων) ενός πίνακα χρησιμοποιώντας την ιδιότητα length.

console.log(cars.length); // Εκτυπώνει: 3

Προσθήκη Στοιχείων

Στο τέλος χρησιμοποιώντας την push()

cars.push("Mercedes");
console.log(cars); // Εκτυπώνει: ["Toyota", "Honda", "BMW", "Mercedes"]

Στην αρχή χρησιμοποιώντας την unshift()

cars.unshift("Ford");
console.log(cars); // Εκτυπώνει: ["Ford", "Toyota", "Honda", "BMW", "Mercedes"]

Αφαίρεση Στοιχείων

Από το τέλος χρησιμοποιώντας την pop()

cars.pop(); // Αφαιρεί το τελευταίο στοιχείο (Mercedes)
console.log(cars); // Εκτυπώνει: ["Ford", "Toyota", "Honda", "BMW"]

Από την αρχή χρησιμοποιώντας την shift()

cars.shift(); // Αφαιρεί το πρώτο στοιχείο (Ford)
console.log(cars); // Εκτυπώνει: ["Toyota", "Honda", "BMW"]

Αναζήτηση Στοιχείων

Μπορείς να βρεις τη θέση ενός στοιχείου στον πίνακα χρησιμοποιώντας την indexOf().

let index = cars.indexOf("Honda");
if (index !== -1) {
    console.log(`Η Honda βρίσκεται στη θέση: ${index}`); // Εκτυπώνει: Η Honda βρίσκεται στη θέση: 1
} else {
    console.log("Η Honda δεν βρέθηκε στον πίνακα.");
}

Δημιουργία Αντιγράφων Πίνακα

Μπορείς να δημιουργήσεις έναν νέο πίνακα από ένα υπάρχοντα χρησιμοποιώντας την slice().

let newcars = cars.slice(0, 2); // Παίρνει τα πρώτα 2 cars
console.log(newcars); // Εκτυπώνει: ["Toyota", "Honda"]

Συνένωση Πινάκων

Μπορείς να συνενώσεις δύο ή περισσότερους πίνακες χρησιμοποιώντας την concat().

let morecars = newcars.concat(["Nissan", "Mazda"]);
console.log(morecars); // Εκτυπώνει: ["Toyota", "Honda", "Nissan", "Mazda"]

Επαναλήψεις σε Πίνακες

Μπορείς να χρησιμοποιήσεις τις μεθόδους forEach() και map() για να επαναλάβεις τα στοιχεία του πίνακα.

  1. forEach(): Εκτελεί μια συνάρτηση για κάθε στοιχείο του πίνακα. 
    cars.forEach((car) => {
    console.log(car);
});
  2. map(): Δημιουργεί έναν νέο πίνακα με τα αποτελέσματα της εκτέλεσης μιας συνάρτησης σε κάθε στοιχείο του αρχικού πίνακα. 
    let lengthcar = cars.map((car) => car.length);
console.log(lengthcar); // Εκτυπώνει: [6, 5, 3]
