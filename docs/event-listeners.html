JavaScript | EventListeners

EventListeners: Χρησιμοποιούνται για την παρακολούθηση ενεργειών του χρήστη και την εκτέλεση κώδικα όταν συμβούν αυτές οι ενέργειες.

1) DOMContentLoaded

Το event DOMContentLoaded ενεργοποιείται όταν το αρχικό HTML έγγραφο έχει φορτωθεί πλήρως και έχει αναλυθεί (parsed) από τον browser, αλλά χωρίς να περιμένει την πλήρη φόρτωση εξωτερικών πόρων, όπως εικόνες και στυλ (CSS).

Όταν τοποθετείτε JavaScript κώδικα στην κεφαλίδα () ενός HTML εγγράφου, υπάρχει πιθανότητα να εκτελεστεί πριν ακόμα φορτωθεί πλήρως το DOM (Document Object Model). Αυτό μπορεί να προκαλέσει σφάλματα αν ο κώδικας προσπαθεί να αλληλεπιδράσει με στοιχεία που δεν έχουν φορτωθεί ακόμα.
Χρησιμοποιώντας το event DOMContentLoaded, εξασφαλίζετε ότι ο κώδικας JavaScript θα εκτελεστεί μόνο αφού το DOM είναι πλήρως διαθέσιμο, αλλά χωρίς να περιμένετε να φορτωθούν και οι εξωτερικοί πόροι (π.χ., εικόνες, αρχεία CSS).

document.addEventListener("DOMContentLoaded", function() {
    // Ο κώδικας εδώ θα εκτελεστεί αφού το DOM έχει φορτωθεί πλήρως
    const button = document.querySelector("button");
    button.addEventListener("click", function() {
        console.log("Το κουμπί έχει πατηθεί!");
    });
});

2) Click EventListeners

Εκτελεί κώδικα όταν ο χρήστης κάνει κλικ σε ένα στοιχείο.

const myButton = document.getElementById("myButton");
myButton.addEventListener("click", function() {
    console.log("Ο χρήστης κάνει κλικ!");
});

3) Mouseover/Mouseout EventListeners

Εκτελούν κώδικα όταν ο χρήστης τοποθετεί το ποντίκι πάνω από ένα στοιχείο ή το απομακρύνει.

const myElement = document.getElementById("myElement");
document.getElementById("myElement").addEventListener("mouseover", function() {
    console.log("Ο χρήστης τοποθετεί το ποντίκι πάνω από το στοιχείο!");
});

document.getElementById("myElement").addEventListener("mouseout", function() {
    console.log("Ο χρήστης απομακρύνει το ποντίκι από το στοιχείο!");
});

4) Mousemove EventListeners

Ενεργοποιείται όταν ο χρήστης κινεί το ποντίκι πάνω από ένα στοιχείο.

document.addEventListener("mousemove", function(event) {
    console.log(`Το ποντίκι μετακινείται: X=${event.clientX}, Y=${event.clientY}`);
});

5) Dblclick EventListeners

Ενεργοποιείται όταν ο χρήστης κάνει διπλό κλικ σε ένα στοιχείο.

document.querySelector("button").addEventListener("dblclick", function() {
    console.log("Ο χρήστης έκανε διπλό κλικ στο κουμπί.");
});

6) Input EventListeners

Εκτελεί κώδικα όταν ο χρήστης πληκτρολογεί ή εκτελεί ενέργειες σε ένα input στοιχείο.

const myInput = document.getElementById("myInput");
myInput.addEventListener("input", function() {
    console.log("Ο χρήστης πληκτρολογεί στο πεδίο εισαγωγής :"+myInput.value);
});

7) Submit EventListeners Listener

Εκτελεί κώδικα όταν ο χρήστης υποβάλλει ένα φόρμα.

      const submitButton = document.getElementById("submitButton");
submitButton.addEventListener("click", function(event) {
    event.preventDefault(); // Αποτρέπει την προεπιλεγμένη συμπεριφορά της υποβολής
    console.log("Ο χρήστης υπέβαλε τη φόρμα!");
});

Η μέθοδος event.preventDefault() χρησιμοποιείται για να ακυρώνει την προεπιλεγμένη συμπεριφορά ενός γεγονότος.

8) Keydown/Keyup EventListeners

Εκτελούν κώδικα όταν ο χρήστης πιέζει ή αφήνει ένα πλήκτρο του πληκτρολογίου.

 document.addEventListener("keydown", function(event) {
    console.log(`Ο χρήστης πίεσε το πλήκτρο με κωδικό ${event.keyCode}`);
});

document.addEventListener("keyup", function(event) {
    console.log(`Ο χρήστης αφήνει το πλήκτρο με κωδικό ${event.keyCode}`);
});

9) Focus EventListeners

Ενεργοποιείται όταν ένα στοιχείο εισαγωγής (input, textarea, select κ.λπ.) λαμβάνει την εστίαση (focus).

document.querySelector("input").addEventListener("focus", function() {
    console.log("Το πεδίο εισαγωγής έχει την εστίαση.");
});

10) Blur EventListeners

Ενεργοποιείται όταν ένα στοιχείο εισαγωγής χάνει την εστίαση.

document.querySelector("input").addEventListener("blur", function() {
    console.log("Το πεδίο εισαγωγής έχασε την εστίαση.");
});

11) Change EventListeners

Ενεργοποιείται όταν η τιμή ενός στοιχείου εισαγωγής αλλάζει και η αλλαγή επιβεβαιώνεται (π.χ., όταν ο χρήστης πατάει "Enter" ή χάνει την εστίαση).

document.querySelector("input").addEventListener("change", function(event) {
    console.log(`Η τιμή άλλαξε σε: ${event.target.value}`);
});

12) Contextmenu EventListeners

Ενεργοποιείται όταν ο χρήστης κάνει δεξί κλικ για να ανοίξει το context menu.

document.addEventListener("contextmenu", function(event) {
    event.preventDefault(); // Απενεργοποίηση του προεπιλεγμένου context menu
    console.log("Το context menu έχει ενεργοποιηθεί.");
});

13) Online και Offline EventListeners

online: Ενεργοποιείται όταν το πρόγραμμα περιήγησης συνδέεται στο διαδίκτυο.
offline: Ενεργοποιείται όταν το πρόγραμμα περιήγησης αποσυνδέεται από το διαδίκτυο.

window.addEventListener("online", function() {
    console.log("Το πρόγραμμα περιήγησης είναι πλέον online.");
});

window.addEventListener("offline", function() {
    console.log("Το πρόγραμμα περιήγησης είναι πλέον offline.");
});

14) Copy, Cut, Paste EventListeners

Αυτά τα γεγονότα ενεργοποιούνται κατά την αντιγραφή (copy), αποκοπή (cut) και επικόλληση (paste) δεδομένων.

document.addEventListener("copy", function(event) {
    console.log("Έγινε αντιγραφή περιεχομένου.");
});

document.addEventListener("cut", function(event) {
    console.log("Έγινε αποκοπή περιεχομένου.");
});

document.addEventListener("paste", function(event) {
    console.log("Έγινε επικόλληση περιεχομένου.");
});

15) Unload EventListeners

Ενεργοποιείται όταν ο χρήστης φεύγει από τη σελίδα (π.χ., κλείνει το παράθυρο ή ανανεώνει τη σελίδα).

window.addEventListener("unload", function() {
    console.log("Ο χρήστης φεύγει από τη σελίδα.");
});

16) Load EventListeners

Ενεργοποιείται όταν η σελίδα ή ένα στοιχείο (όπως μια εικόνα) φορτώνεται πλήρως.

window.addEventListener("load", function() {
    console.log("Η σελίδα φορτώθηκε πλήρως.");
});

17) Unload EventListeners

Ενεργοποιείται όταν ο χρήστης φεύγει από τη σελίδα (π.χ., κλείνει το παράθυρο ή ανανεώνει τη σελίδα).

window.addEventListener("unload", function() {
    console.log("Ο χρήστης φεύγει από τη σελίδα.");
});

18) Wheel EventListeners

Ενεργοποιείται όταν ο χρήστης κυλάει τον τροχό του ποντικιού (scrolling με το ποντίκι).

document.addEventListener("wheel", function(event) {
    console.log(`Ο χρήστης κύλησε τον τροχό του ποντικιού: Δέλτα Y=${event.deltaY}`);
});

19) Animationstart, animationend, animationiteration EventListeners

Αυτά τα γεγονότα ενεργοποιούνται κατά την έναρξη, το τέλος και την επανάληψη μιας CSS animation.

let animElement = document.querySelector(".animated-element");

animElement.addEventListener("animationstart", function() {
    console.log("Η animation ξεκίνησε.");
});

animElement.addEventListener("animationend", function() {
    console.log("Η animation ολοκληρώθηκε.");
});

animElement.addEventListener("animationiteration", function() {
    console.log("Η animation επαναλαμβάνεται.");
});

20) Transitionstart, transitionend EventListeners

Αυτά τα γεγονότα ενεργοποιούνται κατά την έναρξη και το τέλος μιας CSS transition.

let transElement = document.querySelector(".transitioned-element");

transElement.addEventListener("transitionstart", function() {
    console.log("Η transition ξεκίνησε.");
});

transElement.addEventListener("transitionend", function() {
    console.log("Η transition ολοκληρώθηκε.");
});

21) Scroll EventListeners

Ενεργοποιείται όταν ο χρήστης κάνει κύλιση (scroll) στη σελίδα ή σε ένα στοιχείο.

window.addEventListener("scroll", function() {
    console.log("Η σελίδα έχει κάνει κύλιση.");
});

22) Beforeunload EventListeners

Ενεργοποιείται πριν ο χρήστης φύγει από τη σελίδα. Μπορεί να χρησιμοποιηθεί για να εμφανίσει ένα προειδοποιητικό μήνυμα.

// π.χ απο ένα textarea
window.addEventListener("beforeunload", function(event) {
    event.preventDefault();
    event.returnValue = ''; // Απαραίτητο για να δείξει προειδοποίηση στον χρήστη
});
